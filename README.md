# 🪔 Diwali Sales Analysis

This project performs **Exploratory Data Analysis (EDA)** on a Diwali retail sales dataset to uncover customer purchasing behavior, spending patterns, and top-performing product categories.

## 📊 Key Insights
- Dataset after cleaning: **11,251 rows × 13 columns**
- Married women aged **26–35 years** from **Uttar Pradesh, Maharashtra, and Karnataka** working in **IT, Healthcare, and Aviation** sectors are the top spenders
- Popular categories: **Food, Clothing, and Electronics**
- Visual breakdowns by **Gender, Age Group, Occupation, State, and Product Category**

## 🛠️ Tools & Libraries
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook
